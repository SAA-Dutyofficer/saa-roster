📖 System Guide

Getting Started

1. Dashboard — set month, year, section and department then Save
2. Staff — add all staff with their shift pattern, group, rotation and cycle start
3. Leaves — add leave entries. Working days only are shown per entry in the table below
4. ⚡ Generate Roster — auto-builds the full month from patterns and leaves
5. Roster → Shift Summary — click any cell to manually add staff to a shift. Added staff appear alongside the original group with their group in brackets e.g. [Staff - F]. Coverage and Per Person update automatically.
6. Coverage — check for gaps. Fix by going back to Roster and overriding cells
7. Print / XLS / PDF — exports match exactly what you see on screen

Shift Patterns

  • 3W3X — 3 days on, 3 days off. Default rotation M→A→N
  • 6W4X — 6 days on, 4 days off
  • General Duty — Mon–Thu = G, Fri/Sat/Sun = off
  • Manual — set each day individually by clicking cells in Per Person view
  • Custom — define your own work/off day count

Rotation Options

  • DEFAULT — full M→A→N→repeat rotation
  • ROT-MA / ROT-AN / ROT-MN — two-shift rotations
  • ROT-MMA — 2 Morning blocks then 1 Afternoon block per cycle
  • ROT-MAA — 1 Morning then 2 Afternoon blocks per cycle
  • ROT-MMN — 2 Morning then 1 Night per cycle
  • ROT-MNN — 1 Morning then 2 Night per cycle
  • ROT-AAN — 2 Afternoon then 1 Night per cycle
  • ROT-ANN — 1 Afternoon then 2 Night per cycle
  • LOCKED M/A/N/G — always that shift, never rotates

Leave Rules

  • All leave except Military: only working/duty days counted — off days show as x
  • Military Service — ALL calendar days counted including weekends and off days
  • Leave table shows each working day as a separate row so you can see exactly which days are counted

Overrides (Shift Summary)

  • Click any shift cell to manually add specific staff to that shift for that day
  • Original group squares remain — added staff appear alongside in dashed brackets showing their original group e.g. [Obaid - A]
  • Added staff's rosterData is updated so Coverage, Per Person and exports all reflect the change
  • Clear an override to revert to auto group squares only

Exports

  • 📊 Shift Summary XLS — exact match to Shift Summary roster tab: group squares, manual overrides, Staff by Group, Leave table
  • 📋 Per Person XLS — exact match to Per Person roster tab: colored shift grid, Staff by Group, Leave table
  • 📄 PDF / 🖨 Print — prints the current active view with all sections included

User Roles

  • Admin — full access including user management
  • Supervisor — add/edit staff and leaves, edit roster, view coverage, export
  • Staff — Roster tab only, read-only. Can view Shift Summary or own schedule
